Program specification

Program PDFgetX2 should run in the same way on all supported platforms as the same source is used. The look and feel of the GUI may vary slightly. The program offers flexible choices of data corrections. Essentially all processing parameters are accessible from the GUI. Most intermediate data during correction steps can be directly visualized for quick problem diagnosis. The S(Q) and G(r) data are automatically saved by default.
